Bienvenue sur MonSite

Un site pour apprendre le JavaScript événementiel.

À propos de ce site

Ce site est un exercice du TP2 R2.12. Il sert de support pour apprendre à manipuler le DOM avec JavaScript.

Vous allez coder le menu burger, cette modale, et l'accordéon FAQ en JavaScript pur.

À propos

Ce site démontre les interactions JavaScript de base : ouverture/fermeture de composants, gestion d'état avec des classes CSS, et accessibilité clavier.

Tout le JavaScript est dans un fichier script.js séparé. Le CSS gère l'apparence, le JS gère le comportement.

Questions fréquentes

Le DOM (Document Object Model) est la représentation de la page Web en mémoire. JavaScript peut le lire et le modifier pour créer des interactions.

Parce que le JavaScript gère le comportement et le CSS gère l'apparence. En utilisant classList, on garde chaque responsabilité à sa place.

C'est la méthode qui permet de dire au navigateur : "quand tel événement arrive sur cet élément, exécute ce code". C'est le cœur du JavaScript événementiel.

Probablement parce que le fichier script.js n'est pas encore lié, ou parce que le JavaScript n'est pas encore écrit. C'est votre mission !
